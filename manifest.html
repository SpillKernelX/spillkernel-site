Manifest

Useful software should still make sense after the handoff.

SpillKernel is built around tools that install cleanly, expose clear operating surfaces, and leave people with systems they can actually run and maintain.

What matters

Operator clarity beats spectacle.

The work is biased toward Windows-first tooling, practical interfaces, and delivery paths that do not assume perfect conditions. Setup is treated as part of the product. Maintenance is treated as part of the design.

A useful system shows its state, keeps escape routes visible, and reduces the ceremony between the user and the task. That principle applies equally to terminal tools, desktop utilities, and operational web software.
